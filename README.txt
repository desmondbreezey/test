Файлы тестового задания.
Моделирование производилось в бесплатной версии симулятора Modelsim 10.4b, размещенной на сайте компании Altera. Синтез модуля crossbar производился
в бесплатной версии Quartus Prime 15.1.

Описание содержимого:
1) Папка Quartus2
Содержит полный проект для Quartus Prime. Файл crossbar_project_constrains.sdc содержит констрейны для проекта.

2) test_vectors
Содержит тестовые вектора для проведения моделирования.

3) crossbar
Модуль коммутатора

4) slave
Функциональная модель slave'a.

5) master
Функциональная модель master'a.

6) test_top_level
Топ-уровень для моделирования. Содержит два мастера, коммутатор и два слэйва.

7) tb
Тестбенч для проведения моделирования работы топ-уровня (test_top_level).

8) screen
Скриншот с результатами синтеза в Quartus Prime.
